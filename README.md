# fitness-performance-analysis
Fitness data analysis project exploring workout behavior, calorie expenditure, hydration trends, and exercise patterns using Google Sheets dashboards and visualizations.
## Analysis Process
The dataset was explored and analyzed using pivot tables, filters, and visualizations in Google Sheets. Age groups were categorized into ranges to support demographic comparisons, and average-based metrics were used to compare workout behavior, calorie expenditure, hydration trends, and experience levels across participants.
